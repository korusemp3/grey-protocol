# Blackfiles / Grey Protocol — краткая техсводка проекта
Что это
Blackfiles - внутренняя OSINT/досье-карта в рамках сайта Grey Protocol.

Это не просто страница со списком персонажей, а интерактивная карта связей с:
- карточками сущностей
- линиями связей
- pop-up с кратким досье
- уровнями угроз
- внешними сущностями (external)
- возможностью ручного перетаскивания карточек
